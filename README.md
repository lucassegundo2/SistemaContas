Sistema de Contas em C

Projeto desenvolvido em linguagem C utilizando arquivos binários.

Funcionalidades:

- Cadastro de clientes
- Consulta por número da conta
- Atualização de saldo
- Encerramento de conta
- Listagem de clientes
- Releitura do arquivo com rewind()

Conceitos utilizados

- Struct
- Arquivos binários
- fread()
- fwrite()
- fseek()
- rewind()
